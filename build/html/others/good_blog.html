7. 推荐博客和链接¶

把自己在网络上浏览的一些感觉比较不错的博客和链接记录下来。

7.1. openstack¶

  • int32的博客 ：该博客好几篇文章对nova/openstack/调试/源码阅读有很精彩的阐述。

  • fishcried博客 ：openstack网络基础、Linux命令、调试等干货。

  • koala bear ：nova/keystone。

  • apporc’s blog ：nova/oslo/python干货很多，尤其是在创建虚机的一篇博客中，使用dot标记语言绘制整个流程，非常的赞！

  • 世民谈云计算 ：强烈推荐，nova/neutron/cinder/行业分析等，IBM前工程师，创业公司技术总监。

  • popsuper1982 ：nova/neutron等，创建虚机的100个知识点！

  • 孔令贤 ：heat PTL, core member。

    另外，孔令贤有如下两个博客：尤其从他的iteye博客可以看出，他从2012年接触OpenStack时，还没有任何基础，甚至连Python都不会。可是一年半后，他已经是OpenStack core member了。并且16年通过技术移民新西兰，多么励志的故事！

    [1]

    http://blog.csdn.net/lynn_kong
    [2]

    http://lynnkong.iteye.com/

  • 徐超笔记 ：专注于OpenStack测试，neutron项目等！

  • 右转 ：neutron部分好文！

  • 360-ops ：奇虎360运维团队。

7.2. neutron网络系列¶

  • neutron网络基础系列 ：neutron项目相关概念介绍！
  • neutron网络架构系列 ：neutron项目架构简单介绍!
  • Blogs about cloud ：老外的博客，专注于云计算，虚拟化等， 对网络虚拟化有很多精彩的描述，英文写的也很是通俗易懂！强烈推荐。
