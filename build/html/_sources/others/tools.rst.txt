软件和工具
###########


有时电脑会重装，这里把自己电脑中用到的一些
重要工具记录下来，供装机参考。


windows装机必备
+++++++++++++++

- 浏览器：Chrome浏览器，Firefox浏览器

  其中Firefox浏览器推荐的开发插件有 httpfox、RESTClient。
  360浏览器不推荐，但是有些插件也还不错，包括购物小蜜、360抢票等。

- 搜狗输入法  
- Foxmail邮箱客户端
- avast杀毒软件
- 360WIFI、360云盘(弃用)

  360云盘好坑，把资料转移完毕后彻底弃用
  
- 翻墙：SaveVPN、蓝灯
- UltraISO光盘刻录
- ApowerSoft录屏软件
- 抢票：12306bypass、12306订票助手.net
- HP打印扫描驱动
- 有道词典、网易云音乐
- 眼睛卫士
- 搜狗截图工具


开发相关
+++++++++

- jetbrain Pycharm
- Notepad++、Sublime Text
- putty、xshell/xftp 单文件版本、SecureCRT
- teamviewer 远程工具
- Beyond Compare
- samba共享工具
- vmware workstations
- processon在线绘图、graphviz
- sphinx doc、有道云笔记、cmd Markdown(弃用)、马克飞象(弃用)

  原来习惯使用 Markdown 记笔记，现在已经转到 sphinx doc，因此之前使用的
  markdown软件已经弃用，文章陆续迁移中。  
  另外，自己在网上发现一些好的文章，或者重要的博客、网站等，自己都会通过
  有道云笔记的收藏功能收藏起来，以免下次搜索之苦！
